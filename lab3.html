Step 1

Preparations
  • For Lab 2 I followed the instructions on learn.unity.com, a website for learning Unity. There are some step-by-step tutorials on how to do different simple applications, following which will teach you to get acquainted with Unity. What I found interesting is that I could connect with my unity ID and track my progress.
  • First thing I did was Create a new scene. For that in the top right corner I opened Layout (Screenshot 16) and clicked Default (17).
  • Then opened File -> New Scene -> Basic URP, a default option -> Create.
  • After we created a scene, we opened File -> Save As and saved our scene in the Scenes folder and gave it a name, in my case, I’ll name it “Minigame”.

Step 3

Create Plane
  • Open GameObject (or “Plus” symbol in Hierarchy) -> 3D Object -> Plane. As a result a plane appears on the scene. It’s automatically selected.
  • Then in the inspector on the right panel I changed the name of the Plane. I called it Ground, as it is where I will later place all the other objects.
  • After I in Transform I clicked on 3 dots -> Reset to place our ground in the center of the scene with the coordinates (0,0,0).
  • I changed the size of the Plane to 2x2 by changing the Scale. There are several ways to scale an object. Usually it’s more comfortable to use a Scale Tool, which is usually next to a scene. However when I know the exact object size I need, I prefer typing it to Transform -> Scale in the Inspector.
  • By clicking “Plus” in hierarchy -> 3D Object -> Sphere, I created a sphere, reset its position to 0 and renamed it to Player. It’s gonna be my player.
  • To see the sphere better I clicked on the F key and the camera moved closer to my Player. To make it seem like the sphere is standing on a plane I moved a sphere up on half of its size, in my case on 0.5 by Y axes.

Step 5

Set Lightning
  • All Unity Scenes come with the sky box and directional light to represent the sun. To change the color of directional light I selected it in the hierarchy and reviewed it in the inspector. There In section Light -> Emission -> Color I changed color to just white to see all the objects in default color.
  • In order to make a Player stand out I decided to color it. For that I created a Materials folder By clicking on Plus in Project section -> Folder and named a folder “Materials”.
  • After left-clicking on the Material folder -> Create -> Material I created material for the background. I selected Background material and changed Base Color to darker grey and then assigned it to a plane by dragging a background material to a Plane on the scene.
  • Then the same way I created a material for a Player, I picked a nice blue color, increased smoothness a little bit to add shyness and applied this Player material to a Sphere. At the end I have a blue colored ball standing on the grey ground.
  • In order to make a ball move, first, we need to add physics. For that I added a Rigitbody component to my Player as it’s explained on the screen.

Step 6
